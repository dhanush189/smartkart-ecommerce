About SmartKart

A simple ecommerce admin system built for clear product management.

SmartKart brings the core admin workflow into one friendly interface: secure signup, login, product creation, catalog filtering, profile updates, and image uploads.

Start Managing Contact

Project Focus

This application is designed to be practical, smooth, and easy to extend with customer-facing pages, cart features, order management, and payment workflows later.

Purpose

Give admins an easy place to manage ecommerce products without a confusing interface.

Security

Admin accounts use OTP verification and hashed passwords for safer access.

Design

Clean modern design, readable spacing, and smooth hover effects keep the app approachable.
